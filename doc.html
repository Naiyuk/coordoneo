Installation

Installer les dépendances

composer install

Configuration

Editer le fichier config.php se trouvant dans le répertoire /web. Modifier les valeurs en lien avec la base de données par les vôtres (l5, l6, l7, l8)

Créer la base de données

Pour créer la base de données et les tables, utiliser le fichier donnees.sql situait à la racine du projet, via votre logiciel de gestion de bdd MySql ou simplement via la commande MySql

SOURCE donnees.sql

Ce fichier comprend aussi des données fictives pour la démo

Configuration serveur

Htaccess

S'il n'est pas déjà présent, créer un fichier .htaccess dans le répertoire /web comme ci-dessous

RewriteEngine On

RewriteCond %{REQUEST_FILENAME} !-f

RewriteRule ^(.*)$ app.php [QSA,L]

Apache

Pour le serveur web, j'ai créé un fichier de Virtual host qui "redirige" dans le répertoire /web du projet

Lancer l'application !

Après avoir fait la configuration serveur, pour pouvez-vous rendre sur l'url votreDomaine/
